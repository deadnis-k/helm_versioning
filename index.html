Welcome to My Helm Charts Repository!

This repository hosts Helm charts that you can use to deploy applications to Kubernetes.

How to Use

To add this repository to your Helm client, run the following command:

        helm repo add my-repo https://yourusername.github.io/my-helm-charts/charts

Then, run:

        helm repo update

Now you can install any available charts from the repository, like this:

        helm install my-release my-repo/mychart

  • MyChart - A sample Helm chart.
